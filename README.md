# 🚀 User Directory Dashboard

A modern **React-based User Directory Dashboard** that displays users in a clean, interactive table with search, sorting, and detailed profile views.

---

## 📌 Live Features

* 🔍 **Search Users** by name or email (client-side filtering)
* 🔃 **Sort Users** by:

  * Name (ascending/descending)
  * Company (ascending/descending)
* 📊 **Table Layout UI** with modern dark theme
* 👤 **User Detail View** with complete information
* ⚡ **Fast & Responsive UI** using Vite
* 🎨 **Glassmorphism Design** with smooth animations

---

## 🖥️ Preview

> Displays user data from API in a structured table with interactive UI

## 🔗 API Used

* https://jsonplaceholder.typicode.com/users

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Routing:** React Router DOM
* **Styling:** CSS (Custom + Glass UI)
* **State Management:** React Hooks

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/UjgiriKarthik/BuyerForeSight-Frontend-.git
cd user-directory-dashboard
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Run the project

```
npm run dev
```

👉 Open in browser:

```
http://localhost:5173/
```
